Excel Basics: Tables, Formulas, Cell References, and File Control

A beginner guide to cells, tables, formulas, references, sorting, filtering, saving, and verification in Excel.

Published: 2026-07-22Last updated: 2026-07-22
In this guide
  1. Fundamentals
  2. Syntax
  3. Steps
  4. Business examples
  5. Common errors

Fundamentals

Excel is a spreadsheet application for data entry, calculation, analysis, and reporting. Separating input data, calculations, and presentation produces more maintainable workbooks.

Syntax

=A2+B2
=SUM(C2:C10)
=$A$1*B2

Steps

  1. Use unique column headings in the first row.
  2. Keep one data type in each column.
  3. Begin formulas with = and verify cell references.
  4. Maintain one continuous range before sorting or filtering.
  5. Use Save As or version control while preserving source data.

Business examples

  • summarizing sales by month and owner
  • filtering a customer list
  • displaying status from dates and deadlines
  • creating print reports separately from raw data

Common errors

  • using merged cells inside data tables
  • mixing numeric values and text-form numbers
  • overwriting formula cells with values
  • creating uncontrolled copies with no clear current version
Key point

After building the formula, test zero values, blanks, duplicates, and boundary cases.

Frequently asked questions

Where do I enter the formula?

Select the result cell and enter the formula beginning with an equals sign. Then verify the highlighted references and the formula bar.

What if the result does not update?

Check automatic calculation, whether referenced cells are numeric, and whether external links or filters affect the result.

Can I copy the formula to other rows?

Yes, but verify that relative and absolute references behave as intended before filling the formula down.
