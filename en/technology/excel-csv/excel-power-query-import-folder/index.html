Combine Monthly Files from a Folder with Excel Power Query

Import recurring CSV or Excel files with the same structure and refresh the combined data when new files arrive.

Published: 2026-07-22Last updated: 2026-07-22
In this guide
  1. When folder import helps
  2. Setup workflow
  3. Source-file standards
  4. Refresh checks

When folder import helps

When locations or months produce files with the same layout, a folder data source avoids repeated copy-and-paste work.

The files should have consistent headers, data types, and table structure.

Setup workflow

  1. Create a folder containing only files intended for consolidation.
  2. Choose Folder as the data source.
  3. Select a sample file and define transformation steps.
  4. Remove unnecessary columns and set names and data types.
  5. Load the result to a table or the data model.

Source-file standards

  • Keep header names and structure consistent.
  • Exclude titles, subtotals, and notes from the transaction area.
  • Do not place temporary or backup files in the import folder.
  • Include a month or location code in each filename.

Refresh checks

  • Confirm the new files are in the correct folder.
  • Look for errors caused by renamed columns.
  • Validate date and numeric data types.
  • Check that duplicate files were not imported twice.
  • Reconcile counts and totals to source files.
Key point

The reliability of automation depends more on source-file standards than on complex query logic.
