Highlight Overdue and Unresolved Work with Excel Conditional Formatting

Use formula rules based on due date and status to identify records that need action.

Published: 2026-07-22Last updated: 2026-07-22
In this guide
  1. Define the purpose of each colour
  2. Overdue formula
  3. Setup
  4. Rule management

Define the purpose of each colour

Conditional formatting should identify exceptions that require action, not decorate the entire workbook. Limit rules to overdue, due soon, unassigned, or missing required data.

Do not rely on colour alone; keep a status or warning label in the data.

Overdue formula

This example assumes due date in column C, status in column D, and data beginning on row 2.

=AND($C2<TODAY(),$C2<>"",$D2<>"Done")

Setup

  1. Select the full data range.
  2. Create a rule based on a formula.
  3. Set relative and absolute references using the first data row.
  4. Choose a restrained fill or icon.
  5. Test complete, blank, and boundary-date cases.

Rule management

  • Avoid many overlapping rules on the same range.
  • Review priority and stop-if-true behavior.
  • Provide a legend for colour meanings.
  • Treat formula errors as a separate control.
  • Use an Excel table so the rule expands with new rows.
Key point

Too many low-priority colours make the important warnings harder to see.
