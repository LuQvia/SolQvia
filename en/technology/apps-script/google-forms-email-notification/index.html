How to Send Google Forms Responses by Email Automatically

Use an installable form-submit trigger and MailApp to send a structured email whenever a new response arrives.

Published: 2026-07-22Last updated: 2026-07-22
In this guide
  1. Understand the workflow
  2. Setup steps
  3. Sample code
  4. Operational checks

Understand the workflow

The standard pattern is: Google Forms writes a response to a linked spreadsheet, an installable form-submit trigger runs an Apps Script function, and MailApp sends the notification. This gives you control over recipients, subject lines, formatting, and conditions.

The built-in Forms notification can be enough for simple cases. Apps Script is useful when the message must contain selected answers, route to different teams, or follow custom rules.

Setup steps

  1. Open the spreadsheet linked to the form and launch Apps Script from Extensions.
  2. Paste the sample code and replace the recipient address.
  3. Create an installable trigger for onFormSubmit and select the form-submit event.
  4. Authorize the script once, submit a test response, and verify the subject and message body.

Sample code

e.namedValues returns answers keyed by question title. It is often easier to maintain than relying only on fixed column numbers.

function onFormSubmit(e) {
  const recipient = 'info@example.com';
  const values = e.namedValues;
  const lines = Object.keys(values).map(key => {
    return `${key}: ${values[key].join(', ')}`;
  });

  MailApp.sendEmail({
    to: recipient,
    subject: 'New Google Forms response',
    body: lines.join('\n')
  });
}

Operational checks

  • Installable triggers run as the account that created the trigger.
  • Apps Script email quotas apply, so this is not designed for high-volume campaigns.
  • Review recipients and sharing rules when form responses contain personal or confidential data.
  • Retest the script after changing question titles or form structure.
Key point

Test blank answers, multiple-choice responses, and long text before using the workflow in production.
