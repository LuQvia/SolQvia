How to Write with ChatGPT: Separate Outline, Draft, and Editing

A controlled writing workflow that separates outline, drafting, fact checking, and editing when using ChatGPT.

Published: 2026-07-22Last updated: 2026-07-22
In this guide

  2. Inputs to prepare
  3. Core workflow
  4. Prompt template
  5. How to review the output
  6. More advanced use

Objective

For reliable writing, separate objective definition, outlining, drafting, verification, and editing instead of requesting a final document in one step.

The purpose is not to delegate the entire responsibility to AI, but to transform work into a format that is easier for a person to review and decide on.

Inputs to prepare

  • Purpose and audience
  • Required facts and approved sources
  • Length, tone, and prohibited wording
  • Final approver and pre-publication checks

Core workflow

  1. Define the purpose and reader in one sentence.
  2. Generate only an outline and review overlap and gaps.
  3. Provide source information for each section and draft incrementally.
  4. Verify figures, names, and quotations against primary sources.
  5. Edit for readability, terminology, legal risk, and brand voice.

Prompt template

Adapt this template to the task and your review process.

Objective: [what the text must achieve]
Audience: [who will read it]
Source facts: [confirmed information]
Do not: invent facts, quotations, or unsupported certainty
Output: outline first, then draft after review

How to review the output

  • Whether facts or experiences absent from the source were invented
  • Whether quotations preserve the original meaning
  • Whether the same point is repeated in different words
  • Whether the reader’s next action is clear

More advanced use

  • Store editorial rules and prohibited wording in a Project
  • Generate alternatives and score them with explicit criteria
  • Use post-publication search and support data in the next revision
Key point

For repeat use, standardize the input form, approver, storage location, and review date.

Frequently asked questions

How much can be automated?

AI is useful for structuring, drafting, generating options, explaining, and assisting checks. Final approval and exception handling should remain assigned to a person.

How can I improve accuracy?

Specify the objective, audience, input definitions, required output format, prohibited content, and a good example.

What should I do when the output is wrong?

Check the evidence, split the input conditions, and rerun the task. Repeated errors may indicate a source-data or process-definition problem rather than a prompt problem.
