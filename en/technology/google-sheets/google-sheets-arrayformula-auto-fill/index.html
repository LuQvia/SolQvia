Auto-fill Google Sheets Calculations with ARRAYFORMULA

Use one formula to calculate new form-response or transaction rows automatically without copying formulas manually.

Published: 2026-07-22Last updated: 2026-07-22
In this guide
  1. What ARRAYFORMULA solves
  2. Basic example
  3. Reliable design
  4. When not to use it

What ARRAYFORMULA solves

Ordinary formulas are often copied down row by row. ARRAYFORMULA can expand one expression across many rows, reducing missing formulas in append-only data such as form responses.

The output cannot expand into occupied cells, so calculated columns should not also be manual-entry columns.

Basic example

This calculates column B multiplied by column C only when column A contains a record.

=ARRAYFORMULA(IF(A2:A="","",B2:B*C2:C))

Reliable design

  1. Fix the header and first data row.
  2. Choose one required field to detect active rows.
  3. Do not allow manual entries in the spill column.
  4. Use a realistic maximum row when full-column references become heavy.
  5. Validate the original formula before hiding errors.

When not to use it

  • Rows require individually customised formulas.
  • Manual overrides must be preserved in the same column.
  • Many complex full-column calculations cause slow recalculation.
  • A downstream system requires fixed values.
Key point

Automatic expansion works best when input and calculation columns are clearly separated.
