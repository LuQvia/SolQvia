How to Highlight Deadlines and Exceptions with Google Sheets Conditional Formatting

Create clear rules for overdue tasks, missing values, duplicate entries, and missed targets.

Published: 2026-07-22Last updated: 2026-07-22
In this guide
  1. What conditional formatting does
  2. Overdue-row example
  3. Useful formulas
  4. Configuration checks

What conditional formatting does

Conditional formatting changes visual styling without changing the underlying value. It can highlight overdue tasks, blanks, missed targets, or duplicates.

Too many colors reduce clarity. Use a small, documented set of states such as warning, review, and complete.

Overdue-row example

This assumes column C contains the deadline and D contains status. Apply the rule to A2:D to highlight the full row.

=AND($C2<TODAY(),$D2<>"Complete",$C2<>"")

Useful formulas

  • Blank: =A2=
  • Duplicate: =COUNTIF($A:$A,A2)>1
  • Below target: =$E2<$F2
  • Due today: =$C2=TODAY()
  • Due within three days: =AND($C2>=TODAY(),$C2<=TODAY()+3)

Configuration checks

  • Write the formula relative to the top-left cell of the applied range.
  • Use $ to lock columns or rows intentionally.
  • Check whether multiple rules conflict.
  • Do not communicate status by red and green alone; include text or icons.
Key point

Conditional formatting supports monitoring, but critical deadlines should also have notification and ownership controls.
