Columns and Variance Formulas for an Inventory Count Template

Record book quantity, physical count, quantity variance, value variance, recount, and adjustment history.

Published: 2026-07-22Last updated: 2026-07-22
In this guide
  1. Template structure
  2. Recommended columns
  3. Core calculations
  4. Input controls

Template structure

An inventory template should preserve variance and review history, not only the physical count.

Reference item name, unit, and cost from a master table while counters enter the item ID and physical quantity.

Recommended columns

  • Count ID, count date, and location.
  • Item ID, item name, and unit.
  • Book quantity, physical quantity, and variance.
  • Unit cost, value variance, and reason.
  • Recount, reviewer, and adjustment date.

Core calculations

  • Quantity variance = physical quantity minus book quantity.
  • Value variance = quantity variance multiplied by unit cost.
  • Recount variance = recount quantity minus book quantity.

Input controls

  1. Select item IDs from validation choices.
  2. Populate name and cost from the item master.
  3. Allow input only in physical-count fields.
  4. Highlight material differences.
  5. Protect approved rows.
Key point

Keep the original variance after adjustments so the count remains auditable.
