広告収益とアフィリエイトを両立させるページ設計

情報記事では広告、比較・申込意図の強い記事ではアフィリエイトを優先する考え方を解説します。

公開日: 2026-07-22最終更新: 2026-07-22
この記事の内容
  1. 収益源が競合する理由
  2. ページタイプ別の方針
  3. 測定する指標
  4. テストの進め方

収益源が競合する理由

ディスプレイ広告をクリックした読者は、アフィリエイトリンクや自社サービスの申込みへ進まない可能性があります。数円から数百円の広告収益のために、高単価の成約機会を失うことがあります。

そのため、全ページに同じ広告量を設定せず、検索意図とページの役割で分けます。

ページタイプ別の方針

  • 用語解説・一般的な操作方法：広告を主収益にしやすい。
  • 比較・ランキング・個別レビュー：アフィリエイトCTAを優先する。
  • 無料計算ツール：入力欄と結果付近には広告を置かない。
  • 自社サービス案内：外部広告を原則置かない。
  • アクセスは多いが案件がない記事：広告で未収益PVを回収する。

測定する指標

  • ページ別広告収益
  • アフィリエイトリンクCTR
  • 成約率・承認率
  • 1,000セッション当たり総収益
  • 離脱率・エンゲージメント
  • Core Web Vitals

テストの進め方

  1. 広告導入前の4週間程度の基準値を保存します。
  2. 情報記事の一部だけへ広告を導入します。
  3. 広告収益だけでなく、アフィリエイト収益を含む総収益を比較します。
  4. CTA付近の広告、全画面広告、追従広告は個別に検証します。
  5. 収益が増えても表示速度や信頼性が大きく下がる場合は戻します。
ポイント

最適化の基準は広告RPMではなく、ページの目的を損なわない総合収益です。

掲載内容は一般的な情報です。実際の仕様、料金、制度、営業時間などは公式情報をご確認ください。
