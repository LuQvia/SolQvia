ExcelのFILTER関数の使い方｜条件に合う行を自動抽出

FILTER関数で一条件・複数条件の行を抽出し、結果なし、スピル、元データ更新へ対応する方法を解説します。

公開日: 2026-07-22最終更新: 2026-07-22
この記事の内容
  1. 基本
  2. 書式
  3. 作成手順
  4. 実務例
  5. よくあるエラー

基本

FILTER関数は条件に合う行や列を動的配列として返します。元データが変わると結果も更新されますが、出力先に値があるとスピルエラーになります。

書式

=FILTER(A2:D100,D2:D100="対応中","該当なし")
=FILTER(A2:D100,(B2:B100="東京")*(D2:D100>=100000),"該当なし")

作成手順

  1. 抽出する表全体を決めます。
  2. 条件列と条件値を確認します。
  3. 出力先の下と右に十分な空白を確保します。
  4. 結果なしの場合の表示を指定します。
  5. 元データの行追加へ対応するためテーブル化を検討します。

実務例

  • 対応中案件だけを別表へ表示する
  • 店舗と売上額の複数条件で抽出する
  • 期限超過一覧を自動生成する
  • 入力表から印刷対象だけを表示する

よくあるエラー

  • スピル範囲に値や結合セルがある
  • 条件範囲と抽出範囲の行数が違う
  • ANDとORの演算記号を取り違える
  • 抽出結果へ直接入力しようとする
ポイント

数式を完成させたら、0件、空白、重複、境界値を含むテストデータで検算します。

よくある質問

数式はどこに入力しますか？

結果を表示したいセルを選び、半角のイコール記号から数式を入力します。入力後は参照範囲を色や数式バーで確認します。

結果が更新されない場合はどうしますか？

計算方法が自動になっているか、参照セルが数値として認識されているか、外部リンクやフィルターの影響がないかを確認します。

数式を他の行へコピーしてよいですか？

可能ですが、相対参照と絶対参照が意図どおりか確認してからコピーしてください。

掲載内容は一般的な情報です。実際の仕様、料金、制度、営業時間などは公式情報をご確認ください。
