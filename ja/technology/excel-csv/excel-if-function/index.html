ExcelのIF関数の使い方｜条件分岐と入れ子を複雑にしない方法

IF関数の基本、文字・数値・日付の条件、空白処理、複雑な入れ子を避ける設計を解説します。

基本

IF関数は条件がTRUEかFALSEかで異なる値を返します。条件が増えすぎると読みづらくなるため、判定表、IFS、AND、ORなどへ分ける判断が必要です。

書式

=IF(C2>=100,"達成","未達")
=IF(D2="","未入力",D2)
=IF(AND(B2>=80,C2="完了"),"対象","対象外")

作成手順

  1. 判定したい条件を日本語で書き出します。
  2. TRUEの場合とFALSEの場合の戻り値を決めます。
  3. セル型が数値、文字、日付のどれか確認します。
  4. 最小のIFを作り、境界値でテストします。
  5. 条件が3段以上になったら表や別列への分解を検討します。

実務例

  • 売上目標の達成・未達表示
  • 期限超過の判定
  • 入力済み・未入力の表示
  • 複数条件を満たす対象者の分類

よくあるエラー

  • >=と>の境界条件を取り違える
  • 数字と文字列を比較する
  • 空白と0を同じ扱いにする
  • 多数のIFを入れ子にして修正不能になる
ポイント

数式を完成させたら、0件、空白、重複、境界値を含むテストデータで検算します。

よくある質問

数式はどこに入力しますか？

結果を表示したいセルを選び、半角のイコール記号から数式を入力します。入力後は参照範囲を色や数式バーで確認します。

結果が更新されない場合はどうしますか？

計算方法が自動になっているか、参照セルが数値として認識されているか、外部リンクやフィルターの影響がないかを確認します。

数式を他の行へコピーしてよいですか？

可能ですが、相対参照と絶対参照が意図どおりか確認してからコピーしてください。
