ExcelのSORT関数の使い方｜元データを崩さず自動並べ替え

SORT関数で元データを残したまま昇順・降順・複数列の表示を作り、スピルエラーを防ぎます。

公開日: 2026-07-22最終更新: 2026-07-22
この記事の内容
  1. 基本
  2. 書式
  3. 作成手順
  4. 実務例
  5. よくあるエラー

基本

SORT関数は元データを並べ替えず、並べ替えた結果を別範囲へ動的配列として表示します。作業用一覧やランキングに向いています。

書式

=SORT(A2:D100,4,-1)
=SORT(A2:D100,{2,4},{1,-1})

作成手順

  1. 並べ替え対象の表と列番号を確認します。
  2. 昇順1または降順-1を指定します。
  3. 結果を表示する空白範囲を確保します。
  4. 同順位時の第二キーが必要か確認します。
  5. 元データの追加範囲をテーブルで管理します。

実務例

  • 売上の高い順に担当者を表示する
  • 期限が近い順のタスク一覧を作る
  • 店舗名順と売上降順の複合並べ替え
  • 元の入力順を保持したまま表示用一覧を作る

よくあるエラー

  • 列番号を元シートの列番号と混同する
  • 日付や数値が文字列で正しく並ばない
  • スピル範囲が塞がれている
  • 並べ替え結果を固定値と誤認する
ポイント

数式を完成させたら、0件、空白、重複、境界値を含むテストデータで検算します。

よくある質問

数式はどこに入力しますか？

結果を表示したいセルを選び、半角のイコール記号から数式を入力します。入力後は参照範囲を色や数式バーで確認します。

結果が更新されない場合はどうしますか？

計算方法が自動になっているか、参照セルが数値として認識されているか、外部リンクやフィルターの影響がないかを確認します。

数式を他の行へコピーしてよいですか？

可能ですが、相対参照と絶対参照が意図どおりか確認してからコピーしてください。
