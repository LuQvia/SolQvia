Excel・CSV

Excel Power Query入門｜読込・整形・更新を自動化する

Power QueryでCSVやExcelを読み込み、列型、不要行、結合、更新手順を記録する基本を解説します。

公開日: 2026-07-22最終更新: 2026-07-22

  1. 基本
  2. 書式
  3. 作成手順

基本

Power Queryは外部データの読み込みと整形手順を記録し、更新時に同じ処理を再実行する機能です。元データを直接編集せず、再現可能な集計前処理を作れます。

書式

基本の流れ
データ取得 → 列型の設定 → 不要行削除 → 分割・結合 → 読み込み → 更新

作成手順

  1. 元データの保存場所と形式を固定します。
  2. データの取得から対象ファイルへ接続します。
  3. 先頭で列名とデータ型を設定します。
  4. 不要行、空白、列分割、結合を順番に追加します。
  5. サンプル総計と行数を確認して読み込みます。

実務例

  • 毎月同じ形式のCSVを更新する
  • フォルダー内の複数ファイルを結合する
  • マスタ表と実績表をキーで結合する
  • 住所やコードを区切り文字で分割する

よくあるエラー

  • 自動判定のデータ型をそのまま信用する
  • ファイル名や列名変更で更新が失敗する
  • 途中ステップで必要列を削除する
  • 更新後の件数と総計を検算しない
ポイント

数式を完成させたら、0件、空白、重複、境界値を含むテストデータで検算します。

よくある質問

数式はどこに入力しますか？

結果を表示したいセルを選び、半角のイコール記号から数式を入力します。入力後は参照範囲を色や数式バーで確認します。

結果が更新されない場合はどうしますか？

計算方法が自動になっているか、参照セルが数値として認識されているか、外部リンクやフィルターの影響がないかを確認します。

数式を他の行へコピーしてよいですか？

可能ですが、相対参照と絶対参照が意図どおりか確認してからコピーしてください。
