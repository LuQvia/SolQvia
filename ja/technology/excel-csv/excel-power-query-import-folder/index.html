Excel・CSV

Excel Power Queryでフォルダー内の月次ファイルをまとめる

同じ列構成のCSVやExcelファイルをフォルダーから一括取得し、毎月の追加分を更新操作だけで統合します。

公開日: 2026-07-22最終更新: 2026-07-22

向いている業務

店舗別や月別に同じ形式のファイルが増える場合、コピー貼り付けを繰り返すより、Power Queryでフォルダーをデータ源にすると更新しやすくなります。

列名、データ型、ファイル内の表構成が共通していることが重要です。

作成手順

  1. 統合対象だけを保存する専用フォルダーを作ります。
  2. データからフォルダーをデータ源として選択します。
  3. サンプルファイルを指定して変換ルールを作ります。
  4. 不要列の削除、列名、データ型を整えます。
  5. 読み込み先をテーブルまたはデータモデルに設定します。

ファイル側のルール

  • 見出し名と列順をできるだけ統一する。
  • 小計、タイトル、注記を明細範囲へ入れない。
  • 一時ファイルやバックアップを同じフォルダーへ置かない。
  • ファイル名に対象月や店舗コードを含める。

更新時の確認

  • 追加ファイルが対象フォルダーにあるか。
  • 列名変更によるエラーがないか。
  • 日付と金額のデータ型が正しいか。
  • 重複ファイルを二重に取り込んでいないか。
  • 件数と合計を元ファイルと照合したか。
ポイント

自動統合の成否は、Power Queryの式よりも入力ファイルの標準化で決まります。
